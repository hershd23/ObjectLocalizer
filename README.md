# ObjectLocalizer
An object localizer made for the Flipkart grid ML challenge. We achieved 86% accuracy(approx), and had a rank of 124 in the contest.

## Getting Started
These notbooks were made and imported from Google Colab. The following notebooks would run as it is in colab. Find the instructions for running the same code in your machine below in this readme.
